# 🏥 ArogyaMitra – Healthcare Assistance Platform

ArogyaMitra is a **web-based healthcare platform** designed to help users discover nearby hospitals, doctors, and healthcare services.  
The application integrates **location-based services and interactive maps** so users can quickly find medical facilities and essential health services.

This project aims to **improve healthcare accessibility** by providing a simple interface to search healthcare providers and view their locations.

---

## 🚀 Features

- 🔍 Search nearby **hospitals and healthcare centers**
- 🗺️ Interactive **map-based location visualization**
- 🏥 View **available healthcare services**
- 📱 **Responsive design** for desktop and mobile
- ⚡ Fast and intuitive **user interface**
